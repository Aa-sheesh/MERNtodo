1. Download zip folder and unzip.

2. Open the unzipped folder with VS Code and open terminal (Ctrl+`) and write:

```cd backend
    npm i
    npm run start
```

3. `cd .. cd frontend npm i npm run dev `
4. Press Ctrl + o
